# 🏨 نظام إدارة الفنادق - Hotel Management System

## تنفيذ ✅

تم بناء نظام إدارة فندق كامل باستخدام Angular مع جميع الميزات المطلوبة.

## المكونات المعمارية 🏗️

### البنية العامة

## الميزات المطبقة 📋

### 1️⃣ تسجيل الدخول ✅
- واجهة تسجيل دخول باللغة العربية
- حفظ حالة المصادقة
- حماية المسارات (Auth Guard)
- إضافة التوكن تلقائياً للطلبات HTTP

### 2️⃣ لوحة التحكم ✅
- إحصائيات الفندق (الغرف، العملاء، الحجوزات)
- نسبة الإشغال اليومية
- الرسوم البيانية لعرض الأداء
- آخر الحجوزات

### 3️⃣ إدارة الغرف ✅
- إضافة/تعديل/حذف الغرف
- عرض جميع الغرف في جدول
- فلترة حسب النوع والحالة
- تعديل حالة الغرف (متاحة/مشغولة/صيانة)

### 4️⃣ إدارة العملاء ✅
- إضافة/تعديل/حذف العملاء
- عرض تفاصيل العميل
- البحث عن العملاء
- تتبع سجل الحجوزات

### 5️⃣ إدارة الحجوزات ✅
- إنشاء حجوزات جديدة
- حساب السعر تلقائياً
- تعديل حالة الحجز
- فلترة حسب الحالة

### 6️⃣ تقويم الحجوزات ✅
- عرض الحجوزات على التقويم
- التنقل بين الأشهر
- ألوان تشير لحالة كل حجز:
  - 🟢 أخضر = مؤكد
  - 🟡 أصفر = تم تسجيل الدخول
  - ⚪ رمادي = تم تسجيل الخروج
  - 🔴 أحمر = ملغي

### 7️⃣ البحث المتقدم ✅
- البحث في الغرف والحجوزات
- فلترة حسب التاريخ
- فلترة حسب نوع الغرفة
- فلترة حسب السعر

### 8️⃣ تصميم حديث ✅
- دعم اللغة العربية (RTL)
- ألوان جذابة وتناسقية
- تصميم متجاوب (Mobile-friendly)
- وضع الظلام/النهار

## التقنيات المستخدمة 🛠️

- **Angular 21** - الإطار الأساسي
- **Angular Signals** - لإدارة الحالة
- **Angular Material & Tailwind CSS** - للتصميم

- **RxJS** - لإدارة التدفقات
- **SCSS** - للتصميم المتقدم
- **HttpClient** - للتعامل مع API
