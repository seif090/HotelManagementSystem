# 🏨 نظام إدارة الفندق - Hotel Management System

## نظرة عامة
نظام إدارة فندق متكامل بواجهة مستخدم حديثة وسهلة الاستخدام، مُصمم خصيصاً لإدارة جميع جوانب الفندق من الغرف، الحجوزات، العملاء، والتقويم.

## المميزات الرئيسية 🌟

- ✅ **لوحة تحكم تفاعلية** - نظرة شاملة على أداء الفندق
- ✅ **إدارة الغرف** - إضافة، تعديل، حذف الغرف مع فلترة متقدمة
- ✅ **إدارة العملاء** - قاعدة بيانات شاملة لعملاء الفندق
- ✅ **إدارة الحجوزات** - تتبع وتعديل حالة الحجوزات بسهولة
- ✅ **تقويم تفاعلي** - عرض الحجوزات على التقويم مع إمكانية السحب والإفلات
- ✅ **بحث متقدم** - فلترة الغرف والحجوزات بناءً على عدة معايير
- ✅ **دعم اللغة العربية** - واجهة RTL بالكامل
- ✅ **وضع الظلام/النهار** - تبديل سريع بين الوضعين
- ✅ **إشعارات فورية** - تنبيهات بالعمليات المختلفة
- ✅ **تصميم متجاوب** - يعمل بكفاءة على جميع الأجهزة

## التقنيات المستخدمة 🛠️

- **Angular 21** (Standalone Components)
- **Angular Material** & **Tailwind CSS**
- **Angular Signals** - لإدارة الحالة
- **FullCalendar** - لعرض التقويم
- **RxJS** - لإدارة التدفقات
- **SCSS** - للتصميم المتقدم

## التشغيل المحلي 🚀

### المتطلبات الأساسية
- Node.js (v18 أو أعلى)
- npm (v9 أو أعلى)

### خطوات التثبيت

1. **استنساخ المشروع:**
```bash
git clone <repository-url>
cd hotel-management-system
```

2. **تثبيت الحزم:**
```bash
npm install
```

3. **تشغيل بيئة التطوير:**
```bash
npm start
# أو
ng serve
```

4. **فتح المتصفح:**
```
http://localhost:4200
```

### بيانات التجربة 🧪

لتجربة النظام فوراً:
- **البريد الإلكتروني:** `admin@hotel.com`
- **كلمة المرور:** `admin123`

## استخدام التطبيق 💻

### 1. تسجيل الدخول
- أدخل بريدك الإلكتروني وكلمة المرور
- اضغط على "تسجيل الدخول"

### 2. لوحة التحكم
- عرض إحصائيات الفندق (الغرف، العملاء، الحجوزات)
- عرض نسبة الإشغال والرسوم البيانية
- عرض آخر الحجوزات

### 3. إدارة الغرف
- إضافة غرف جديدة بجميع تفاصيلها
- تعديل أو حذف الغرف الموجودة
- تغيير حالة الغرفة (متاحة/مشغولة/صيانة)
- فلترة الغرف حسب النوع والحالة

### 4. إدارة العملاء
- إضافة عملاء جدد
- تعديل بيانات العملاء
- عرض وتتبع سجل الحجوزات لكل عميل

### 5. إدارة الحجوزات
- إنشاء حجز جديد مع حساب السعر التلقائي
- تعديل حالة الحجز (مؤكد/ملغي/تم تسجيل الدخول/تم تسجيل الخروج)
- حذف الحجوزات

### 6. التقويم
- عرض جميع الحجوزات على التقويم
- تتبع توافر الغرف حسب التاريخ
- ألوان تشير لحالة كل حجز

### 7. البحث المتقدم
- البحث عن غرف متاحة في تواريخ محددة
- فلترة حسب السعر والنوع
- البحث في الحجوزات حسب العميل أو التاريخ

## الـ API 🔌

يتم ربط التطبيق بـ Backend Node.js عبر الـ HTTP Interceptor لإضافة التوكن تلقائياً إلى جميع الطلبات.

### بنية الـ API المتوقعة:

```
POST   /api/auth/login         # تسجيل الدخول
GET    /api/rooms              # جميع الغرف
POST   /api/rooms              # إضافة غرفة جديدة
PUT    /api/rooms/:id          # تعديل غرفة
DELETE /api/rooms/:id          # حذف غرفة

GET    /api/customers          # جميع العملاء
POST   /api/customers          # إضافة عميل
PUT    /api/customers/:id      # تعديل عميل
DELETE /api/customers/:id      # حذف عميل

GET    /api/bookings           # جميع الحجوزات
POST   /api/bookings           # إضافة حجز
PUT    /api/bookings/:id       # تعديل حجز
DELETE /api/bookings/:id       # حذف حجز
```
